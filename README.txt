Le dossier est constitué des fichiers de code suivants :
rapport.pdf décrivant chaque prédicat;
tBoxABox.pl représentant la TBox et la ABox;
partie1.pl, partie2.pl, partie3.pl correspondant aux implémentations des parties 1, 2 et 3 de l'énoncé;
programme.pl qui lance la résolution.

Pour lancer le programme dans le terminal avec l'interprete swipl, il suffit de charger tous les fichiers avec la commande:
[partie1,partie2,partie3,tBoxABox,programme].
puis d'exécuter la commande programme(Abe). qui mettra en route le démonstrateur.
Il faut ensuite choisir quelle type de proposition démontrer (1 ou 2) et entrer la proposition.
Le programme affiche ensuite toutes les étapes de la résolution jusqu'à sa conclusion.
